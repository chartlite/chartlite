Chartlite supports multiple data input formats for maximum flexibility

Original Format: DataPoint[]

Classic array of objects with x and y properties

new LineChart('#chart1', { data: [ { x: 'Jan', y: 4200 }, { x: 'Feb', y: 4800 }, { x: 'Mar', y: 5200 } ] });

Simple Array New

Just pass an array of numbers - perfect for quick prototypes

new BarChart('#chart2', { data: [30, 45, 38, 52, 60] }); // Auto-generates x values: 0, 1, 2, 3, 4

Column-Oriented Recommended

Intuitive format inspired by data frames and spreadsheets

new AreaChart('#chart3', { data: { x: ['Q1', 'Q2', 'Q3', 'Q4'], y: [45000, 52000, 48000, 61000] }, curve: 'smooth' });

Series-First Recommended

Define data structure upfront - great for complex datasets

new LineChart('#chart4', { data: { series: [ { name: 'Revenue', dataKey: 'revenue', color: '#3b82f6' } ], data: [ { month: 'Jan', revenue: 4200 }, { month: 'Feb', revenue: 4800 }, { month: 'Mar', revenue: 5200 } ] }, curve: 'smooth' });

Series-First with Colors

Colors from series definition are auto-applied

new BarChart('#chart5', { data: { series: [ { name: 'Sales', dataKey: 'sales', color: '#10b981' // Auto-applied! } ], data: [ { product: 'Product A', sales: 320 }, { product: 'Product B', sales: 280 }, { product: 'Product C', sales: 245 } ], xKey: 'product' // Optional: specify x-axis key }, orientation: 'horizontal' });

Column-Oriented (Multi-Series)

For single-series charts, uses the first series

new AreaChart('#chart6', { data: { x: ['Mon', 'Tue', 'Wed', 'Thu', 'Fri'], y: { temperature: [22, 24, 26, 25, 27], humidity: [60, 58, 55, 62, 59] // Uses 'temperature' (first series) } }, colors: ['#f59e0b'], fillOpacity: 0.2 });
